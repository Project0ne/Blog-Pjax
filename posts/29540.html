Hexo添加导航页

效果

效果演示

新建页面

首先新建页面，执行下面的命令

hexo new page navigate

修改navigate目录下的index.md的格式

---
title: 导航
date: 2020-10-18 15:19:14
type: "navigate"
layout: "navigate"
---

在主题配置文件中添加导航

# main menu navigation url and icon
# 配置菜单导航的名称、路径和图标icon.
menu:
  Index:
    url: /
    icon: fas fa-home
  标签:
    url: /tags
    icon: fas fa-tags
  时光轴:
    url: /archives
    icon: fas fa-hourglass-half
  留言板:
    url: /contact
    icon: fas fa-comments
  关于:
    url: /about
    icon: fas fa-user-circle  
  友情链接:
    url: /friends
    icon: fas fa-address-book
  Select:
    icon: fas fa-life-ring
    children:
      - name: 导航
        url: /navigate
        icon: fas fa-location-arrow
      - name: 相册
        url: /galleries
        icon: fas fa-image

添加navigate.ejs页面
